# vigor-website

The new landing page for Vigor.

## Development

Uses [Gatsby.js](https://www.gatsbyjs.org/docs/) for static site rendering.

```bash
npm i

# local development
npm start

# creating static sites
npm run build
```

## Transations

Translations must be provided in `src/i18n/resources.json`.
